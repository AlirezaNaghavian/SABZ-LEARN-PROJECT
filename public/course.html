عنوان دوره گویای همه چی هست اما نیازه برخی موارد گفته بشه تا بتونید با دید بهتری تو این دوره شرکت کنید و هدف دوره رو بهتر بتونید درک کنید.

افراد بعد از یادگیری Css نیازه که مینی پروژه های مختلفی رو بزنن و تا بتونن با مهارت های خامی که با Css دارن یکسری خروجی های رو بگیرن اما نکته مهمی که وجود داره و اکثر افراد بهش توجه نمیکنن، نوع پروژه و نحوه پیاده سازی اون هست، مورد اول که ما داخل دوره ازش استفاده میکنیم اینه که میاییم و مینی پروژه هایی که زیاد توی بازار کار استفاده میشن رو پیاده سازی میکنیم و نحوه کد نویسی اصولی و استاندارد رو یاد میگیرید، پس شما بعد از دیدن این دوره مهارت Html,Css  شما بهتر میشه و دید خیلی حرفه ای تری رو نسبت بهش بدست میارید.

اما تفاوت این دوره با سایر دوره ها چیه؟

اصلی ترین تفاوتی که این دوره با سایر دوره ها داره، اینه که توی این دوره مثل دوره Tailwind من دیزاین اختصاصی برای شما دانشجو ها در نظر گرفتم، که تمامی مینی پروژه هایی که قراره کار کنیم حدودا 10 الی 20 مینی پروژه خوب، توی Figma طراحی شده و قراره به صورت خیلی حرفه ای کار با طرح و نحوه پیاده سازیشو با Html,Css یادبگیرید. دقیقا عین یک پروژه حرفه ای توی بازار کار که به شما دیزاین تحویل میدن و کد ازتون میگیرن، پس یکی از تفاوت های اصلی که این دوره با سایز دوره ها داره، اینه که از خودمون پروژه نمیزنیم، بلکه از روی دیزاین اختصاصی که مختص دانشجو های این دوره هزینه شده و طراحی شده میزنیم.

پیشنیاز های این دوره چیه؟

برای اینکه بتونید این دوره رو ببینید نیازه که شما دوره های Html و  Css و Flexbox رو کامل مشاهده کرده باشید و مشکلی توی مباحث خود این دوره ها نداشته باشید، سپس با خیال راحت برای اینکه مشکلاتتون رفع بشه و بتونید از سطح آماتور و مبتدی در بیایید، این دوره رو مشاهده بفرمایید.

ممکنه توی برخی از پروژه های از Js هم استفاده کنیم که اگر بلد هم نباشید مشکلی نیست من کامل براتون توضیح میدم و شما میتونید اون بخش از کد Js رو کپی کنید، مهم Html و  Css هست که شما خودتون باید کداشو بزنید اما به هرحال برای اینکه پروژه هامون انعطاف بیشتری داشته باشن بعضا از Js هم استفاده میکنیم تا کارمون حرفه ای تر هم بشه.

بعد از دیدن دوره به چه سطحی میرسم؟

شما اگه این دوره رو به خوبی ببینید و پروژه هاشو انجام بدید، بعد از دیدن این دوره سه هیچ از بقیه جلوترید، به دلیل اینکه در همین ابتدای کار اصولی و حرفه ای پیش رفتید، نحوه کار با طرح رو یادگرفتید و تونستید طرح رو به کد تبدیل کنید، کد نویسی اصولی رو یادگرفتید و خیلی از بخش ها و مینی پروژه های پرکاربرد بازار رو تونستید پیاده سازی کنید و دیگه توی مراحل بعدی طراحی قالب برای شما عین آب خوردن میشه، شما الان آماده هستید که وارد مرحله بعدی یادگیریتون یعنی طراحی قالب حرفه ای و فریمورک های Css بشید
